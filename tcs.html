TCS Digital & Prime Syllabus

1. TCS Digital Syllabus

Estimated Package: ₹7 LPA

Aptitude + Logical Reasoning

  • Time, Speed, Distance
  • Time and Work
  • Permutation & Combination
  • Probability
  • Ratio & Proportion, Averages, Mixtures
  • Profit and Loss, Number System
  • Data Interpretation, Puzzles
  • Seating Arrangements, Series, Syllogisms

English (Verbal Ability)

  • Reading Comprehension
  • Synonyms/Antonyms
  • Grammar Correction
  • Sentence Completion
  • Error Spotting, Para Jumbles

Programming Logic (MCQs)

  • Data Types, Operators, Loops, Conditions
  • Functions, Recursion, Arrays, Strings
  • Pointers (C/C++), OOPs Concepts
  • Time and Space Complexity
  • Basic DS: Stacks, Queues

Coding Round (2 Questions)

  • Arrays and Strings
  • Recursion, Searching & Sorting
  • Sliding Window, Hashing, Two Pointers
  • Basic Dynamic Programming

2. TCS Prime / Innovator Syllabus

Estimated Package: ₹8–12+ LPA

Includes All of TCS Digital Syllabus

Advanced Coding (DSA)

  • 3 Medium to Hard Problems
  • Graphs, Trees, Tries
  • Backtracking, Greedy, Bit Manipulation
  • Sliding Window, HashMap/Set

Core CS Subjects

  • DBMS: Normalization, Indexing, SQL Joins
  • OS: Deadlocks, Scheduling, Concurrency
  • CN: OSI Layers, TCP/IP, DNS, HTTP
  • System Design: Basic LLD, Scalability Concepts

Case Studies / Innovation Thinking

  • Solve Real-World Problems with Algorithms
  • Justify Data Structure Choices
  • Mini Design Round or Technical Scenario
